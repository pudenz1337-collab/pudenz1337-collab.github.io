▸ How cloud sync works
Your full FuelStrong state is saved to Cloudflare KV (a fast key-value database) every time you log food, check in, or tap "Sync Now". The data is tied to your Worker URL — any device that uses the same Worker URL shares the same data.

Sync Now — pushes your current data to the cloud
Load from Cloud — pulls the latest saved state (useful after switching devices)

Your data also stays in localStorage as a fast local cache, so the app works even if you're offline.
